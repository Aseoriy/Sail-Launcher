An open-source game/app manager with script/mod support, and cloud services with a pretty yummy design.

Library & Emulation Import

Automatically imports your Steam library with 1 click. Fetches cover photos, banners, and descriptions, and supports retro emulator systems with custom parameters.

Native Cloud Save Sync

Direct API integration for backing up and restoring save files with Google Drive, OneDrive, or Dropbox. Auto-syncs launcher configurations. Tokens are encrypted using Electron's native safeStorage.

Workshop & Mod Downloader

An in-app browser interface for community mods. Downloads files anonymously from SteamCMD and extracts `.zip`, `.rar`, and `.7z` folders automatically into your game directories (please work).

Themes & Micro-Animations

It has a frosted glass ui with custom spring animations, background waves, pulse, or rippling flows. Personalize tile shapes (Squircle, Pill, Circle) and select from 10+ built in themes, you can also use a seperate ui from an older version of the launcher.

Custom Pre/Post Scripts

Automate companion programs (trainers, overlay screens, Discord) and trigger custom Batch (`.bat`) or PowerShell (`.ps1`) scripts that run on game launch and end on closure events.

Discord RPC & Web Sources

Broadcast active gaming states, playtime, and graphics to friends. Link your favorite modding and cataloging websites to navigate them smoothly inside the launcher's built-in web container.
