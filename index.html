Configure how games are downloaded and installed.

Runs the repack's setup.exe silently into the install folder, then deletes the leftover setup/.bin files and adds the finished game automatically. The install step is CPU-heavy and may show one Windows permission prompt to approve.

Tells the repack installer to deselect its optional tasks, so it won't re-run the DirectX/Visual C++ installers or open the FitGirl website after installing. Turn this off if a game fails to launch because it needs a runtime you don't have. (Some repacks force a runtime regardless — that's controlled by the repack, not the launcher.)

📖 Setup Instructions
Google Drive Setup Guide
  1. Go to the Google Cloud Console.
  2. Create a project, search for Google Drive API, and click Enable.
  3. Go to OAuth consent screen, select External, and configure it. Add the scope: .../auth/drive.file.
  4. Under Credentials, click + Create Credentials -> OAuth client ID.
  5. Select Application type: Web application.
  6. Under Authorized redirect URIs, add: http://localhost:53232/callback.
  7. Copy the generated Client ID and paste it below.
  8. ⚠️ If you get "Access Blocked / Error 403: access_denied": Go back to the OAuth consent screen page in Google Console, and under Test users, click + Add Users and enter your Google email address. Alternatively, click Publish App under Publishing status to set the app to Production.
Microsoft OneDrive Setup Guide
  1. Go to the Azure Portal (under App registrations).
  2. Click New registration. Set Supported account types to Accounts in any organizational directory and personal Microsoft accounts.
  3. Under Redirect URI, select Web and enter: http://localhost:53232/callback.
  4. Click Register, copy the Application (client) ID from the Overview page, and paste it below.
Dropbox Setup Guide
  1. Go to the Dropbox App Console.
  2. Click Create app. Select Scoped access and Full Dropbox access.
  3. Under the Settings tab, set OAuth 2 Redirect URIs to: http://localhost:53232/callback.
  4. Under the Permissions tab, enable: files.content.write, files.content.read, and files.metadata.read, then click Submit.
  5. Copy the App key (Client ID) and App secret (Client Secret) and paste them below.
