Sail Launcher v5.0.0 Changelog

Biggest update yet! Almost everything got touched — new UI, a full download system with auto-extraction, FitGirl/DODI repack auto-install, poster tiles, a revamped sidebar, friends list with game presence, and a ton of bug fixes and polish. This one took a while but it's finally here.

Major updates:

Complete glassmorphic UI redesign with blur effects, animated gradients, and glowing accents throughout the launcher. The whole thing just looks way better now.

New Download tab powered by aria2 with multi-source support, live progress tracking, and automatic extraction for ZIP and RAR archives. FitGirl and DODI repacks now auto-install completely silently without you having to babysit the window.

Friends list now shows game presence — you can see what your friends are playing in real time through Discord Rich Presence.

New poster/cover tile layout for the library as an alternative to the classic card view.

Continue Playing banner that surfaces your most recently played game right at the top.



## <a name="launcher-updates"></a>Launcher Updates

**UI Overhaul**

#1 Glassmorphic Redesign:
Rebuilt the entire UI with a glassmorphic design system — translucent panels, backdrop blur, layered depth, glowing hover states, and smooth animated gradients on key elements.
Sidebar, cards, modals, settings panels, and the titlebar all got the new look.
Added a translucency intensity slider in System Settings so you can dial it in however you like.

#2 Sidebar Rework:
Moved the navigation tabs into the sidebar header and repositioned the hamburger (☰) toggle to the top-left of the sidebar so it's always accessible and out of the way.
Sidebar collapse/expand is smooth and the layout handles both states cleanly.

#3 Poster Tile Layout:
Added a new poster/cover view for the library that shows tall cover art instead of landscape cards.
Toggle between card and poster view from the top-right library controls.

#4 Continue Playing Banner:
Most recently played game now shows up as a large banner at the top of the library view with a one-click launch button.

**New Features**

#5 Download Tab with aria2:
New Download tab with a full download manager built on aria2c for fast multi-connection downloads.
Paste any direct link, pixeldrain link, or GoFile link and it downloads, shows live speed + progress, and queues automatically.
A Cloudflare-bypassing proxy handles sites that block direct Electron requests.

#6 FitGirl / DODI Auto-Install:
After a repack finishes downloading and extracting, the launcher automatically runs the installer silently — no clicking through setup windows.
Uses InnoSetup silent flags (`/VERYSILENT /SUPPRESSMSGBOXES /NORESTART /NOCANCEL /SP-`) so the whole process is hands-off.
"Skip Redistributables" toggle in settings skips VC++, DirectX, and other redist installs that you probably already have — saves 5+ minutes per install.
Mute toggle silences the installer process so it doesn't blast audio during installation.

#7 Auto-Extraction:
ZIP and RAR files are automatically extracted after download using 7za and node-unrar-js respectively.
Handles the doubled-filename quirk in some repack archives and cleans up SteamRIP junk files automatically.

#8 Friends List with Game Presence:
Added a friends panel that pulls your Discord friends list and shows their current game status via Rich Presence.

#9 Stronger Ad & Popup Blocker:
Expanded the network-level ad blocker with more sources and added popup suppression in the built-in browser so you don't get redirect modals when browsing download sites.

**Bug Fixes & Polish**

#10 Mods Tab Fix:
The Mods tab was showing a black screen after switching to Cloudflare — Cloudflare's bot detection was flagging Electron's user-agent and `sec-ch-ua` headers.
Fixed by running the Mods webview in an isolated in-memory partition (like incognito) and stripping the Electron-identifying headers before requests go out.

#11 Dropdown Toggle Fix:
Clicking the action dropdown (☰ menu) a second time now closes it instead of immediately reopening it. Switched from a `click` listener to `mousedown` with a container check so the timing works correctly.

#12 Game Card Hover Flicker Fix:
Cards flickering when hovering near the bottom edge are fixed. The card lifts on hover which was moving the cursor off the card and re-triggering the transition in a loop. Added a small exit-transition delay so the card settles before the hover state resets.

#13 Save File Detection UI:
When "Save File Detection" is enabled, the Browse button now shows as "Scan" and actively triggers a save location scan instead of being grayed out.
The Override button now toggles to "Cancel" when clicked (instead of a separate Re-detect button), which is a lot cleaner.

#14 Installer Mute Fix:
FitGirl's setup spawns `setup.tmp` as a child process (not `setup.exe`) — PowerShell was filtering by exact process name so the mute wasn't catching it.
Fixed the filter to match `setup.tmp` and similar extensions so the mute actually works.

#15 Version Bump:
Bumped to v5.0.0 across package.json, index.html version display, and the in-app version constant.
